# NAPE-Noise-Aware-Pose-Estimation-for-Reliable-Re-Identification-under-Severe-Nighttime-Degradation
NAPE is a noise-aware pose estimation framework for nighttime person re-identification, designed to improve robustness under low illumination, noise, and blur.
Currently, only a portion of the pedestrian data from the private extreme weather dataset is displayed here.
The subsequent content will be published after the draft of the paper.
